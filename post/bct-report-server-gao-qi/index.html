📣💨✨BCT Report Server搞起😎

2021-02-08 BCT Report Server

快速上手

环境配置

安装

  1. 安装Telerik Report Designer（Telerik_Reporting_R2_2019_13_1_19_514_DEV.msi）

运行发布

  1. 下载BCTWebReportServer_KendoUI.7z，解压并用VS打开工程
  2. 打开Controllers\ReportsController.cs，修改HostAppId，避免冲突
  3. 发布到文件系统，使用IIS搭建网站
    Congratulations, the BCT Report Server has been built successfully ~

绘制报表

可参考DESIGNING REPORTS

发布报表

  1. 将报表文件拖入文件系统的"Reports"文件夹中
  2. 在Web.config中添加与报表文件一致的数据库连接字符串（数据源界面添加亦可）
    <connectionStrings>
        <add name="???" connectionString="data source=???;user id=???;password=???;Initial Catalog=???;Integrated Security=false" providerName="System.Data.SqlClient" />
    </connectionStrings>

访问报表

  1. 发布地址+“/Report/Main?reportName=”+ 报表名称
