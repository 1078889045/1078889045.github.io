📣💨✨BCT Vue搞起😎

2021-02-07 BCT Vue

快速上手

环境配置

工具配置

HBuilder X

  1. 若出现写代码卡顿情况，可在设置中关闭“代码助手”

vscode

可参考Vue Vben Admin 2.0项目文档

安装

  1. 安装 Node.js，Better be v12.18.4
   # 出现相应npm版本即可
   npm -v

   # 出现相应node版本即可
   node -v

安装依赖

  1. 安装 yarn
   # 全局安装yarn
   npm i -g yarn

   # 验证
   yarn -v # 出现对应版本号即代表安装成功
  1. 依赖安装
    # 安装依赖
    yarn install

ECharts需安装v4.9.0

    yarn add echarts@4.9.0

npm Script

安装依赖

    yarn bootstrap

运行项目

    yarn serve

打包项目

    yarn build

清空缓存后构建

    yarn build:no-cache

项目配置项

环境变量配置

项目的环境变量配置位于 项目根目录 .env 、.env.development、.env.production
可参考Vue Vben Admin 2.0项目文档，当前项目未使用最新版本，与文档略有不同
